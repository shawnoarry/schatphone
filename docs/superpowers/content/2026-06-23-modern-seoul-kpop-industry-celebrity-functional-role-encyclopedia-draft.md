## 条目定位

K-pop 行业中的名人、制作人、MC、PD、作家、编舞师、A&R、造型团队、宣传团队、品牌方和现场工作人员，是角色事业线与事件线的重要节点。

这类人物可以是真实名人坐标，也可以是 AU 中的同职能角色。真实姓名提供行业参照，具体邀约、合作、评价、冲突、关系和事件以用户设定、当前日程和 AU 发展为准。

功能性角色不一定常驻私人关系。许多角色只在选曲会议、录音、舞蹈排练、电视台录制、综艺通告、品牌拍摄、机场移动、签售现场、舆情处理和公司会议中出现。一次事件可以让他们被短暂提及，也可以让他们进入后续关系网络。

## 1. 综艺 MC 与节目主持坐标

Yoo Jae-suk：刘在锡；中文社区常称刘大神、大神、国民 MC。常驻/代表语境：Running Man、You Quiz on the Block、Pinggyego / DdeunDdeun、Hangout with Yoo、Family Outing、Infinite Challenge 相关国民综艺记忆。适合承接舒适访谈、公开分寸、国民度、善意带话、成熟艺人回顾和新人紧张缓和。

Kang Ho-dong：姜镐童；中文社区常译姜虎东，也可简称虎东。常驻/代表语境：Knowing Bros、New Journey to the West、新西游记系衍生、Kang's Kitchen、Star King、X-Man 相关综艺记忆。适合承接高能量开场、前后辈玩笑、强推进、棚内游戏、夸张反应和嘉宾综艺感测试。

Shin Dong-yup：申东烨；资深 MC 坐标。常驻/代表语境：Amazing Saturday、My Little Old Boy、Immortal Songs、SNL Korea、Witch Hunt、Radio Star 式 talk show 语境。适合承接 talk show、成人向笑点、尖锐但有分寸的提问、棚内控场、艺人反转故事和标题化传播。

Lee Soo-geun：李寿根；中文社区也常译李秀根。喜剧型 MC 坐标。常驻/代表语境：Knowing Bros、New Journey to the West、新西游记系衍生、Kang's Kitchen、2 Days & 1 Night 相关综艺记忆。适合承接游戏规则、即兴反应、身体喜剧、插科打诨、综艺节奏推进和嘉宾临场反应。

Kim Heechul：金希澈；中文社区常称希澈。idol 出身综艺人坐标。常驻/代表语境：Super Junior、Knowing Bros、Weekly Idol、Idol Room、My Little Old Boy、二代 idol 综艺记忆。适合承接前后辈玩笑、饭圈知识、二代 idol 经验、对新人团体的调侃照顾和节目内 idol 语境翻译。

Boom：本名李敏镐；中文社区多直接称 Boom 或 붐。音乐综艺 MC 坐标。常驻/代表语境：Amazing Saturday、Trot / 音乐类综艺、节目规则说明、固定班底反应。适合承接 Amazing Saturday、游戏说明、夸张反应、音乐梗、气氛推进和固定班底反应。

Jang Do-yeon：张度练；谈话与网络综艺 MC 坐标。常驻/代表语境：Salon Drip、Comedy Big League、女艺人访谈、网络谈话内容、颁奖礼与发布会主持。适合承接轻松提问、女性艺人访谈、舒适倾听、机智反应、Salon Drip 式聊天和不强迫嘉宾自曝的访谈。

Lee Young Ji：李泳知；中文社区常称泳知。新世代网络综艺 MC 坐标。常驻/代表语境：No Prepare / My Alcohol Diary、High School Rapper、Show Me the Money、年轻音乐人访谈和短视频传播。适合承接坦率聊天、年轻语感、音乐人关系、喝酒聊天式节目、朋友感访谈和短视频切片。

Park Bo-gum：朴宝剑；中文社区常称宝剑。常驻/代表语境：Music Bank、Music Bank World Tour / 全球特别场、演员型主持、颁奖礼与大型活动主持。适合承接音乐节目 MC、演员型主持、礼貌形象、全球巡演舞台、后台问候和主持成长线。

音乐节目 MC 位常由现役 idol、演员或新人担任。固定 MC、特别 MC、代班 MC 和全球特别场 MC 都可以制造镜头训练、台本压力、前后辈接触、同框热帖、粉丝比较和主持能力评价。

## 2. 综艺制作人与节目班底坐标

Na Young-seok / Na PD：罗暎锡；中文社区常译罗英石、罗PD。常驻/代表语境：2 Days & 1 Night、New Journey to the West、Three Meals a Day、Youn's Kitchen、Jinny's Kitchen、Earth Arcade、The Game Caterers、Channel Fullmoon。适合承接艺人自然反应、长期综艺企划、旅行任务、固定班底信任感和高传播度节目片段。

Kim Tae-ho / TEO：金泰浩；中文社区常称金泰浩 PD。常驻/代表语境：Infinite Challenge、Hangout with Yoo、TEO、Seoul Check-in、My Name is Gabriel、Salon Drip 相关制作生态。适合承接大型项目邀约、特殊企划、话题包装、节目格式创新和明星关系网。

节目 PD 决定企划方向、嘉宾组合、录制节奏、现场规则、补录要求和节目整体风险。PD 可以是邀约者、协调者、现场压力来源，也可以是帮助艺人展示新面貌的机会节点。

节目作家决定问题、游戏、提示卡、台本梗、角色定位和可讲故事。作家会提前向公司收集可谈话题、禁谈话题、成员资料、宣传重点和可公开趣事。

剪辑团队决定正片分量、字幕语气、预告悬念、shorts 切片、缩略图和高光传播。一个表情、一个停顿、一句玩笑或一次沉默，都可能被剪成角色印象。

现场导演、导播、摄像、收音、灯光和字幕人员决定录制顺序、镜头路线、声音状态、棚内反应和现场节奏。角色在节目中的公开形象常由本人反应、节目组设计和后期加工共同形成。

## 3. SM、JYP、YG 与 THEBLACKLABEL 制作坐标

SM 制作体系重视声乐、概念、视觉、编曲层次和表演完整度。Yoo Young-jin / 俞永镇适合作为 SM 声乐、R&B、vocal direction 和早期到四代 SM 声音坐标；代表语境包括 H.O.T.、S.E.S.、BoA、TVXQ、Super Junior、Girls' Generation、SHINee、EXO、Red Velvet、NCT、aespa 等 SM 系谱。Kenzie 适合作为 SM 长期 songwriter / producer 坐标；代表语境包括 BoA、Girls' Generation、SHINee、f(x)、EXO、Red Velvet、NCT、aespa、RIIZE、Hearts2Hearts 等。LDN Noise、Dem Jointz、Moonshine、Ryan Jhun 等名字适合作为 SM 外部合作、实验编曲、强 hook、NCT / aespa 式声音和全球制作营坐标。相关事件可以涉及 demo 筛选、vocal direction、bridge 高音、复杂和声、实验曲风、概念会议和舞台视觉统一。

JYP 制作体系重视训练稳定性、表演完成度、团体能量、全球市场和公司内部制作协作。Park Jin-young / 朴振英 / J.Y. Park 是公司代表制作人坐标；代表语境包括 Wonder Girls、2PM、miss A、GOT7、TWICE、ITZY、NMIXX、JYP 练习与舞台标准。3RACHA 适合作为 Stray Kids 自制作坐标；代表语境包括 Bang Chan、Changbin、Han、强 rap / performance、成员主导创作和全球巡演男团。Collapsedone、Hong Ji-sang、earattack 等可作为 JYP Publishing 与外部制作合作坐标；代表语境包括 TWICE、DAY6、Stray Kids、ITZY、NMIXX 等不同路线的 demo 合作。相关事件可以涉及主打方向、成员自作曲、demo 评审、英文版发行、声乐录音和舞台稳定性。

YG 与 THEBLACKLABEL 制作体系重视强品牌感、hip-hop / performance 气质、时尚识别、低频但高冲击宣发和制作人厂牌色彩。Teddy / 泰迪适合作为 YG / THEBLACKLABEL 代表制作人坐标；代表语境包括 1TYM、BIGBANG、2NE1、BLACKPINK、Jeon Somi、MEOVV、THEBLACKLABEL 女团路线和全球女团声音。Choice37、24、R.Tee、Vince、Future Bounce 等名字适合作为 YG / THEBLACKLABEL 制作团队坐标；代表语境包括 BIGBANG、WINNER、iKON、BLACKPINK、TREASURE、BABYMONSTER、solo 企划和厂牌内部制作协作。相关事件可以涉及黑厂牌感、drop 设计、rap making、时尚造型、全球女团路线、solo 企划和高规格 MV。

## 4. HYBE 与多厂牌制作坐标

Bang Si-hyuk / Hitman Bang：房时赫；HYBE 创始人、制作人和战略坐标。代表语境包括 BigHit、BTS 早期制作、TXT、HYBE 多厂牌扩张、选秀项目和全球市场战略。适合承接公司战略、全球市场、练习体系、制作资源、团体定位和厂牌协作。

Pdogg：HYBE / BigHit 代表制作人坐标。代表语境包括 BTS、BigHit 系主打制作、rap / vocal 混合、成熟团队长期声音和高版税制作人讨论。适合承接 BTS 相关制作语境、主打候选、录音方向、vocal / rap 处理、成熟团队制作和全球市场声音设计。

Slow Rabbit、Supreme Boi、FRANTS、Wonderkid、ADORA 等名字适合作为 BigHit / HYBE 制作资源坐标。代表语境包括 BTS、TXT、ENHYPEN、GFRIEND、LE SSERAFIM、BigHit / Source / Belift 等厂牌声音协作。相关事件可以涉及 demo 打磨、概念曲、成员参与创作、英文市场、舞台情绪和发行节奏。

BUMZU：PLEDIS / HYBE 制作人坐标。代表语境包括 SEVENTEEN、TWS、PLEDIS 系团队制作、Woozi 合作和大型团体声音组织。适合承接团队自制作、vocal team / performance team / hip-hop team 协作、成员创作会议和长期团体声音经营。

ZICO：禹智皓；KOZ 代表制作人坐标。代表语境包括 Block B、个人 solo、KOZ、BOYNEXTDOOR、hip-hop 与大众旋律结合。适合承接制作人厂牌、年轻男团、hip-hop 语境、社媒传播和成员创作能力。

250、FRNK 等名字适合作为 ADOR / NewJeans 声音坐标。代表语境包括 NewJeans、轻量 R&B、UK garage、复古电子、自然感 MV 和生活化概念。相关事件可以涉及年轻听感、低负担 vocal、时尚生活化企划、demo 方向争论和品牌化声音。

HYBE 多厂牌协作容易带来资源比较、制作风格差异、内部竞争、跨厂牌合作、共同练习空间、年末舞台和品牌资源流动。制作人、A&R、创意总监和厂牌负责人可以在同一项目里拥有不同意见。

## 5. 中型公司、自制作团队与外部制作坐标

EDEN / EDEN-ary 是 KQ Entertainment、ATEEZ 和 xikers 相关制作坐标。代表语境包括 ATEEZ、xikers、KQ 男团、强叙事 performance、巡演能量和团队长期声音设计。

Soyeon / 田小娟 是 (G)I-DLE 自制作坐标。代表语境包括 Produce 101 / Unpretty Rapstar 出身、(G)I-DLE 主打制作、女团自我表达、概念主导和成员创作话语权。适合承接歌词方向、公司资源限制下的作品决策和公众对制作能力的讨论。

Ryan Jhun、MonoTree、PRISMFILTER、Mospick、Black Eyed Pilseung 等外部制作人或制作团队可以作为跨公司合作坐标。Ryan Jhun 适合承接 IVE、NCT、SM / Starship 外部合作语境；MonoTree 适合承接 LOONA、ONF、Red Velvet、NCT 等精细 pop 制作语境；PRISMFILTER 适合承接 SEVENTEEN、fromis_9、TWS 与 PLEDIS / HYBE 相关制作营语境；Black Eyed Pilseung 适合承接 TWICE、STAYC 和大众女团 hook 语境。它们适合承接 demo 竞争、外部邀歌、音色适配、团体转型、回归试水和粉丝对制作班底的期待。

外部 songwriter、topliner、编曲师、混音师和母带工程师通常不进入粉丝视野，但会影响歌曲完成度、成员录音压力、part 分配、英语歌词、hook 记忆点和国际发行质感。

## 6. 舞台、编舞与表演制作坐标

Performance director 负责把音乐、成员能力、舞台规模、镜头路线和概念动作整合成可表演内容。这个角色可以出现在编舞会议、练习室复盘、舞台彩排、巡演排练和成员状态调整中。

编舞师和舞团决定 point choreography、dance break、队形、手势、地板动作、双人互动、成员中心点和挑战动作。Bada Lee、Kiel Tutin、Rie Hata、Leejung Lee 等名字可以作为代表性编舞坐标。具体合作以 AU 设定和当前作品为准。

Vocal director 负责录音棚内的音准、情绪、咬字、呼吸、和声、ad-lib、guide vocal 和成员声音适配。它适合制造录音压力、声音成长、主唱高光、成员被肯定或反复重录的事件。

舞台导演、摄像导演、灯光导演和现场音响负责电视台舞台、颁奖礼、演唱会、fan meeting 和 showcase 的呈现。它们可以制造走位修改、镜头补录、耳返问题、灯光失误、ending 镜头、舞台安全和现场复盘。

## 7. 公司内部功能性角色

A&R 负责人连接公司战略、制作资源、艺人定位和市场目标。这个角色可以发起选曲会议、主打竞争、demo 盲听、成员意见征集、海外 songwriter 对接和发行节奏调整。

经纪人负责日程、移动、沟通、保护边界和现场执行。经纪人可以提醒禁谈话题、催促上车、协调电视台入场、处理粉丝距离、转达公司意见和判断成员身体状态。

生活经纪更贴近宿舍、饮食、休息、未成年成员照顾、学校沟通、医院陪同、练习室出入和日常纪律。生活经纪适合承接新人期、练习生期、准出道期和高压行程后的照顾场景。

宣传、公关和法务负责新闻稿、声明、采访口径、舆情监测、恶评取证、品牌沟通和风险控制。舆情事件中，这些角色决定沉默、澄清、道歉、法律警告、行程调整或暂停活动的公开节奏。

粉丝运营负责 fanclub、粉丝平台、签售名单、入场通知、应援规则、会员内容、生日项目和活动反馈。它适合承接粉丝不满、通知失误、签售争议、应援秩序和官方口吻。

造型总监、化妆师、发型师和服装团队决定概念照、打歌服、红毯造型、品牌活动、发色变化和舞台安全。它们可以制造妆造升级、造型争议、临时修衣、发色损伤和粉丝神图。

## 8. 外部生态功能性角色

品牌 PR 负责艺人与品牌之间的形象匹配、活动邀约、红毯站位、礼服、拍摄脚本、合同边界和舆情风险。品牌方可以带来高级资源，也会放大公开形象压力。

娱乐记者负责标题、采访、机场拍摄、红毯提问、发布会报道和新闻切角。记者可以是曝光入口，也可以是压力来源。未经确认的爆料、标题党和断章取义不自动成为事实。

摄影记者、站子、饭拍者和现场 repo 账号负责把线下状态转成图像和文字材料。它们会影响论坛讨论、粉丝情绪、成员状态判断和关系解读。

翻译、通译和海外 coordinator 负责跨语种采访、海外巡演、品牌活动、机场沟通和国际合作。它们适合承接海外日程、语言误会、采访缓冲和跨文化关系。

安保、司机、场馆工作人员和票务执行负责路线、入口、排队、证件、车辆、散场和突发拥挤。线下事件中，这些角色常决定角色能否安全移动、粉丝能否入场、现场是否延误。

## 9. 临时角色与事件角色边界

功能性角色可以按事件临时出现。常见触发包括：选曲会议、录音、月评、舞蹈排练、回归会议、音乐节目预录、综艺录制、品牌拍摄、机场出发、签售、巡演彩排、公司声明、论坛热帖和舆情升级。

真实名人只作为行业坐标。除非用户明确设定，不写现实私人关系、未公开合作、未证实评价、现实争议、合同细节和私密事件。

同一功能位可以使用真实坐标，也可以使用 AU 化名。Mnet 节目 PD、KBS 现场导演、HYBE A&R 负责人、JYP vocal director、THEBLACKLABEL 外部制作人、tvN 综艺作家、品牌 PR、签售现场执行，都可以成为事件角色。

功能性角色的重点是资源、职责、语气和行动边界。它们可以推动事件、提供压力、打开机会、制造误会、保护角色或传递行业反馈，但不自动成为亲密联系人。

可用关键词：国民 MC、综艺 MC、音乐节目 MC、特别 MC、节目 PD、综艺作家、剪辑、A&R、producer、songwriter、topliner、vocal director、performance director、编舞师、造型总监、宣传、公关、法务、粉丝运营、品牌 PR、记者、站子、翻译、安保、司机、现场执行、临时角色、事件角色。
