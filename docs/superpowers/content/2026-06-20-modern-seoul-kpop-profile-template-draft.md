# 现代首尔 K-pop 娱乐圈：角色档案模板草稿

Updated: 2026-06-20

Status: `PROFILE_TEMPLATE_DRAFT`

这是一组面向“现代首尔 K-pop 娱乐圈”世界观的角色档案模板。模板帮助用户填写角色是谁、身处行业哪里、如何被公众看见、如何与他人保持关系。

模板不预设剧情入口，不限制用户自设身份，不限制用户把角色加入现实团体、替代现实成员、改变团体人数或改变团体构成。

## 模板设计原则

1. 模板只提供可填写的身份坐标，不写行业百科解释。
2. 模板字段应帮助角色在私聊、公开动态、日程、粉丝互动和关系记忆中保持一致。
3. 同一角色在当前版本中优先选择一个主模板；副身份、兼职经历、特殊 AU 设定可写入补充字段或自定义字段。
4. 主视角角色、联系人角色和世界内其他角色都可以使用同一套世界专属模板；区别由用户填写的具体值决定。
5. 字段名称在世界设定和角色档案中保持一致，用户看到的是中文字段名。
6. 真实公司、团体、成员、节目、平台都作为 AU 坐标使用；用户填写的改动优先于默认坐标。

## 通用基础字段

以下字段适合放入每个 K-pop 世界专属模板，便于角色在不同场景里保持稳定身份。

### 世界身份定位

类型建议：短文本或单选

填写内容：这个人在 K-pop 世界里的主要身份，例如现役爱豆、练习生、退役艺人、制作人、经纪人、粉丝站站姐、娱乐记者、品牌工作人员、普通粉丝、论坛用户。

### 所属关系

类型建议：短文本

填写内容：所属公司、厂牌、团体、工作室、节目组、粉丝站、媒体账号、品牌方、论坛圈层或个人活动名义。

### 当前阶段

类型建议：单选

可选方向：未公开、练习期、预备出道、出道初期、上升期、稳定活动期、转型期、暂停活动、个人活动期、退圈或幕后期。

### 公开形象

类型建议：标签

填写内容：门面、实力派、综艺感、忙内、队长、制作型、冷感、亲和、清纯、强势、反差、治愈、野心感、低调、神秘、话题体质等。

### 私下状态

类型建议：长文本

填写内容：角色离开镜头后的生活状态、情绪惯性、社交习惯、压力来源、对公众身份的适应程度。

### 作息与活跃时段

类型建议：标签

填写内容：夜猫子、早起型、拍摄期昼夜颠倒、练习室常驻、通勤党、宿舍生活、独居、巡演时差、考试期、回归期高压、休假期低频。

### 沟通习惯

类型建议：长文本

填写内容：私聊长短、是否秒回、是否爱用语气词、是否谨慎截图、是否会突然消失、是否喜欢语音或照片、是否区分熟人和公众语气。

### 关系边界

类型建议：长文本

填写内容：角色对朋友、同事、粉丝、前后辈、合作方、暧昧对象、公开关系和地下关系的边界。

### 舆情敏感点

类型建议：标签

填写内容：恋爱传闻、队内资源、换人争议、学历争议、舞台失误、健康暂停、合约争议、粉圈对立、造型争议、商务冲突、前成员关系、家人曝光。

### 关键关系人

类型建议：人物关联

填写内容：同团成员、队长、经纪人、制作人、前后辈、好友、竞争者、固定合作方、站姐、记者、品牌联系人、家人。

## 模板一：K-pop 通用人物档案

适用对象：不确定具体行业位置的角色，或兼具多个身份的角色。

推荐字段：

- 世界身份定位
- 所属关系
- 当前阶段
- 公开形象
- 私下状态
- 作息与活跃时段
- 沟通习惯
- 关系边界
- 舆情敏感点
- 关键关系人

使用建议：这是默认推荐模板。用户想快速创建角色时先用这一版，再用自定义字段补充特殊设定。

## 模板二：爱豆 / 练习生档案

适用对象：现役爱豆、练习生、预备出道成员、限定团成员、solo 艺人、退役后仍被粉圈讨论的艺人。

推荐字段：

- 所属公司与团体
- 出道或训练阶段
- 队内位置
- 舞台强项
- 镜头人设
- 私下状态
- 练习与行程节奏
- 粉丝关系
- 同团关系
- 前后辈关系
- 恋爱与避嫌边界
- 舆情敏感点

字段说明：

所属公司与团体：公司、厂牌、团名、solo 名义、限定活动身份都可以填写。

出道或训练阶段：练习生、出道组候补、预告公开、出道初期、回归期、solo 期、暂停活动、转型期等。

队内位置：队长、主唱、领舞、rapper、门面、center、忙内、综艺担当、创作成员、ACE、隐藏成员等。AU 中可以自由改动。

舞台强项：唱功、舞蹈、rap、表情管理、直拍、现场稳定、创作、编舞、综艺反应、语言能力、粉丝互动。

粉丝关系：宠粉、距离感强、营业积极、谨慎克制、容易被误读、站姐多、路人盘强、黑粉集中、CP 粉活跃。

## 模板三：幕后 / 公司工作人员档案

适用对象：制作人、词曲作者、A&R、编舞师、舞台导演、造型师、经纪人、生活经纪、宣传、法务、练习生管理、公司普通职员。

推荐字段：

- 所属公司或团队
- 职能领域
- 负责对象
- 资源权限
- 工作节奏
- 职业风格
- 对外话术
- 与艺人的距离
- 公司立场
- 风险处理习惯
- 关键合作人

字段说明：

职能领域：制作、选曲、训练、舞台、视觉、造型、商务、宣传、危机处理、合同、行程、生活管理等。

负责对象：具体团体、艺人、练习生、项目、节目、品牌合作、海外活动或公司部门。

资源权限：能影响选曲、造型、舞台、行程、宣传、预算、人事、对外回应或私人生活管理的程度。

与艺人的距离：严格管理、熟人式照顾、工作边界清楚、私交深、只负责某段行程、长期陪同、曾有冲突。

风险处理习惯：先压消息、先内部确认、先保护艺人、先保护公司、先看舆论、先联系媒体、先联系粉丝端。

## 模板四：媒体 / 平台 / 社群角色档案

适用对象：娱乐记者、节目 PD、综艺编导、采访编辑、摄影记者、媒体账号运营、论坛管理员、平台运营、翻译搬运号。

推荐字段：

- 公开身份
- 信息来源范围
- 内容风格
- 立场倾向

- 爆料边界

## 推荐启用顺序

第一批建议先启用四个模板：

1. K-pop 通用人物档案
2. 爱豆 / 练习生档案
3. 幕后 / 公司工作人员档案
4. 粉丝 / 站子 / 饭圈角色档案

媒体 / 平台 / 社群角色档案和商务 / 品牌合作档案可以作为第二批启用。它们适合更复杂的舆情、商业合作、节目现场和粉丝社区角色。

## 使用边界

1. 模板只提供角色档案字段，不替用户决定剧情、关系或事业走向。
2. 用户可以跳过不需要的字段，也可以新增世界观没有列出的自定义字段。
3. 真实公司、团体、成员、节目和平台都作为 AU 坐标使用，用户写入的改动优先。
4. 角色档案只保存用户选择写入的内容，不自动继承现实人物的私人关系、健康状态、合约状态或争议事件。
5. 同一角色可以拥有副身份，例如现役 idol 兼制作人、站姐兼翻译号、经纪人兼旧同学。副身份写在补充字段里即可。
6. 生日、出道日期、身高、粉丝规模、合约年限等强数值字段，当前不作为必填项。

## 后续整理备注

1. 后续转成正式模板时，字段名称使用中文，不把内部字段标识展示给用户。
2. 模板字段只使用当前稳定支持的类型：短文本、长文本、单选、标签、人物关联。
3. 如果后续增加日期、数字、等级、时间段等更强字段类型，再把生日、出道日期、身高、粉丝规模、合约年限等内容拆成结构化字段。
4. 当前版本不要求用户填写真实人物资料。现实成员信息仍由代表成员坐标百科提供参考，角色档案只保存用户选择写入的 AU 设定。
