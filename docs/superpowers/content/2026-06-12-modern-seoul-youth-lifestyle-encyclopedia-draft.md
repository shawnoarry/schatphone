## 条目定位

本文件补充现代首尔年轻人的日常生活方式。它用于让 K-pop 世界中的角色在工作之外也像真实生活在 2026 年首尔：会聊天、约见、通勤、吃饭、喝咖啡、逛街、拍照、上课、打工、追热点、处理社交压力。

本文件不用于向用户讲解韩国生活常识。它提供 AI 可调用的生活场景、语气、消费习惯、社交方式和日常动作。用户设定拥有最高优先级；用户可以把角色设定成更富裕、更节俭、更宅、更外向、更国际化或完全不同生活方式。

## 1. 手机通信与社交节奏

现代首尔年轻人的日常强依赖手机。常见通信场景包括 KakaoTalk 群聊、Instagram 私信、X / Twitter 实时动态、短视频平台、粉丝平台通知、外卖提醒、打车软件、地图导航、银行转账、便利店会员码和照片分享。

KakaoTalk 是最常见的即时沟通坐标。角色可以通过私聊、群聊、工作群、粉丝运营群、同学群、练习生群、成员群和家人群维持关系。已读不回、回得很慢、只发贴图、突然语气变正式、深夜补消息、群里不说话但私聊解释，都可以成为关系信号。

Instagram 更适合公开形象、限时动态、私信试探、品牌露出和社交圈观察。年轻角色可能会看谁点赞、谁看了限动、谁取消关注、谁发了同款地点、谁在凌晨发歌单。X / Twitter 更适合粉丝实时讨论、站子预览、趋势和跨语种传播。

沉浸作用：AI 可以用通信平台差异塑造关系距离。亲近的人发短句和贴图，工作关系更正式，暧昧关系在已读、限动、点赞和深夜消息里产生张力。普通 Chat 回复仍保持手机短消息，不把平台行为写成已发生事实，除非用户或上下文明确。

可用关键词：KakaoTalk、私聊、群聊、已读、贴图、限时动态、点赞、取消关注、私信、工作群、成员群、粉丝平台通知。

## 2. 见面、约会与社交边界

年轻人见面常发生在咖啡店、饭店、酒吧、练习室附近、汉江边、公园、商场、电影院、快闪店、展览、便利店、地铁站出口和公司楼附近。初次见面通常会确认时间、地铁出口、穿着特征、是否已经到达、是否需要排队和是否要换地点。

朋友社交可以是约饭、喝咖啡、拍人生四格、逛街、散步、看展、看电影、去练歌房、去 PC 房、去汉江边买拉面、便利店夜宵或凌晨路边聊天。关系越亲近，越容易出现“不说目的也能见面”“只发位置”“点同一份外卖”“顺路送你回去”等细节。

恋爱和暧昧关系在 K-pop 语境中更容易受公开形象影响。角色可能避开热门商圈、选择包间、戴帽子口罩、错开出入时间、用朋友名义约见、避开粉丝站和记者活动范围。普通年轻人也可能因为学校、工作、前任、朋友圈或社媒痕迹而保持边界。

沉浸作用：AI 可以用见面方式表现关系阶段。熟人会问“你到几号出口了？”；朋友会说“先去便利店买点东西？”；暧昧对象会在公开场合保持距离；艺人或工作人员会更注意被拍、被认出和行程保密。

可用关键词：地铁出口、咖啡店、人生四格、汉江、练歌房、PC 房、包间、口罩、错开出入、只发位置、顺路、便利店夜宵。

## 3. 饮食、咖啡与便利店生活

首尔年轻人的日常饮食节奏经常被工作、上课、练习、通勤和社交打碎。常见选择包括便利店饭团、三角饭团、杯面、紫菜包饭、炸鸡、炒年糕、鱼饼、烤肉、部队锅、汤饭、拌饭、冷面、沙拉、低糖饮料、蛋白棒和外卖。

咖啡店是高频社交和等待空间。年轻人会在咖啡店见朋友、写作业、等经纪人、等妆发、看消息、拍照、开小会、赶工作。冰美式、拿铁、甜品、限定杯套、生日咖啡馆和品牌快闪，可以进入普通日常。

便利店是夜间、赶行程和独处场景的重要节点。角色可以在便利店买水、能量饮料、冰杯、拉面、便当、创可贴、口罩、充电线、雨伞和小零食。深夜便利店门口、汉江边便利店、公司附近便利店和宿舍楼下便利店，都能承接短对话。

沉浸作用：AI 可以用饮食细节表现角色状态。忙碌角色可能只喝冰美式、吃蛋白棒和便利店饭团；休息日角色可能约烤肉或咖啡；练习后角色可能想吃辣炒年糕又担心管理。食物不应变成长篇介绍，只作为消息里的生活感。

可用关键词：冰美式、便利店、三角饭团、杯面、紫菜包饭、炸鸡、炒年糕、烤肉、外卖、生日咖啡馆、汉江拉面、能量饮料。

## 4. 居住、通勤与城市移动

年轻人在首尔可能住在宿舍、合租房、考试院、one-room、公寓、家里、公司安排的宿舍或临时酒店。K-pop 从业角色还可能长期在公司、练习室、录音棚、沙龙、电视台和机场之间移动。

通勤常使用地铁、公交、出租车、公司车辆、步行和深夜打车。地铁出口、换乘站、末班车、堵车、雨天、雪天、凌晨打车、机场快线和导航截图，都可以成为聊天里的真实动作。艺人角色可能更多使用保姆车或公司车，但仍会受到堵车、停车、粉丝聚集和安保路线影响。

居住空间决定亲密感。宿舍生活容易出现室友、共用冰箱、洗衣、外卖、练习后倒头睡、成员作息不同、经纪人查房和生活经纪提醒。独居角色更容易出现整理房间、忘记倒垃圾、深夜开灯、阳台、快递、邻居噪音和一个人吃饭。

沉浸作用：AI 可以用居住和通勤状态控制回复节奏。角色在地铁上可能断续回复，在公司车里可能压低语气，在宿舍可能怕吵醒室友，在独居房间可以更放松。地点和移动事实仍需要用户或上下文确认，不自动改写当前所在地点。

可用关键词：宿舍、one-room、合租、考试院、地铁、公交、出租车、公司车、保姆车、末班车、换乘、堵车、机场快线、快递、室友。

## 5. 学习、工作与兼职压力

年轻人的生活常被学习、就业、兼职、考试、实习和家庭期待影响。学生角色可能面对大学课程、小组作业、考试周、社团、语言学习、交换项目和毕业压力。普通工作角色可能面对加班、会食、上司消息、合同工、实习期、转正和工资压力。

兼职常见于咖啡店、便利店、餐饮店、服装店、补习机构、活动现场、翻译、拍摄助理、票务、展会、短期宣传和平台运营。K-pop 世界中，兼职和实习可以把普通年轻人带进娱乐产业现场。

练习生、艺人和工作人员也会有学习与职业压力。练习生需要在学校、训练和评价之间切换；海外成员或工作人员可能需要韩语学习；制作人和工作人员需要跟项目截止时间、深夜会议和临时修改共处。

沉浸作用：AI 可以用学习和兼职压力让非艺人角色也有真实生活。角色可能说“我下课再回你”“今天便利店夜班”“明天考试周”“PD 让我改字幕”“公司又临时改会议”。这些信息可以成为关系推进、迟回复和见面困难的原因。

可用关键词：考试周、小组作业、实习、兼职、便利店夜班、咖啡店打工、活动现场、翻译、拍摄助理、加班、会食、转正、韩语课。

## 6. 消费、审美与拍照文化

年轻人的消费常围绕咖啡、外卖、服装、美妆、香水、电子产品、演唱会票、专辑、小卡、周边、快闪店、展览、拍照馆和社媒内容。消费能力差异会影响生活方式，有人追新款，有人精打细算，有人把钱花在演唱会和专辑上。

拍照和社媒展示是日常的一部分。人生四格、咖啡店打卡、展览照片、穿搭照、镜自拍、街拍、限时动态、滤镜、贴纸、背景音乐和位置标签，都可以进入年轻人的交流方式。K-pop 角色会更注意公开照片是否暴露位置、同款、同框和私人关系。

审美趋势可以连接妆造、穿搭、发色、包、鞋、香水、手机壳、耳机和运动品牌。角色不需要每次都描述全套穿搭，但可以用一两个物件表现生活状态和个性。

沉浸作用：AI 可以用消费和拍照习惯制造社交线索。角色看到同款咖啡店、同款鞋、限动背景、展览票根或拍照馆照片，可能产生猜测、亲近、误会或粉丝讨论。不要把同款自动写成恋爱证据，除非用户设定。

可用关键词：人生四格、限时动态、打卡、穿搭、镜自拍、滤镜、位置标签、快闪店、小卡、周边、香水、手机壳、同款、票根。

## 7. 夜生活、酒局与情绪出口

首尔年轻人的夜间生活可以包括下班后约饭、烤肉、酒吧、练歌房、汉江散步、便利店门口聊天、深夜咖啡、夜间出租车、朋友家小聚和独自回家。夜生活不一定是放纵，也可以是压力后的短暂恢复。

会食和酒局在工作关系中常见，但角色可以选择参与、拒绝、提前离开或保持距离。娱乐产业角色还可能因为第二天行程、形象管理、身体状态、未成年保护、公司规定和被拍风险控制夜间活动。

情绪出口可以是散步、听歌、健身、练舞、写歌、看直播、刷论坛、整理小卡、打游戏、看电影、给朋友发长消息或只发一个表情。年轻角色不一定直接说“我很难过”，可能通过沉默、已读、突然发歌、半夜说想吃东西表现状态。

沉浸作用：AI 可以用夜间场景表达压力、亲密和边界。深夜消息可以更私人，但不自动等于恋爱或危险事件。艺人角色夜间外出更需要考虑被拍、明日行程、经纪人提醒和公司规则。

可用关键词：会食、烤肉、酒吧、练歌房、汉江散步、深夜出租车、便利店门口、半夜消息、发歌、沉默、被拍风险、经纪人提醒。

## 8. 作息规律与活跃时段

作息规律描述角色在一天中的活跃窗口、低活跃窗口、消息风格和生活状态。它可以进入人设字段，也可以作为 Chat、论坛、KakaoStory、粉丝平台动态和社交事件的语义参考。作息不等同于实时在线状态；AI 在被调用时根据当前时间、上下文和角色设定表现“像这个时间段的角色”，不自动创造未确认的行动、位置、已发送动态或系统通知。

夜猫子角色在深夜更清醒，回复更私人、更松弛，也更容易刷论坛、写歌、剪视频、复盘练习、整理情绪或在便利店门口停留。白天的夜猫子可能慢热、短回、刚醒、在补觉，或用“我脑子还没开机”一类说法解释状态。夜猫子适合制作人、练习生、创作者、上升期 idol、夜班兼职者和压力型角色。

晨型角色在早上更稳定，常和咖啡、运动、早课、通勤、日程整理、清淡早餐、天气检查和任务确认连接。晨型角色深夜更容易早睡、拒绝酒局、提醒对方早点休息，或把凌晨消息留到早上认真回复。晨型适合自律型 idol、工作人员、学生、健身习惯明确的角色和生活边界强的角色。

行程型角色的活跃时间随练习、预录、沙龙、机场、直播、拍摄、课程和会议变化。消息常出现“上车了”“候场”“妆发中”“等经纪人”“刚下录制”“到酒店了”“马上没电”等短状态。行程型适合 idol、练习生、经纪人、PD、造型师、翻译和活动工作人员。

不规律型角色的睡眠被工作、压力、时差、舆论、考试、兼职或创作周期打乱。回复可能突然很密，也可能消失一段时间后补一句解释。不规律型适合回归期、巡演期、考试周、舆情期、休养转型期和临时项目期角色。

社交低电量型角色在线但不一定愿意聊天。他们可能已读、只回表情、晚些解释、发限动但不回私信、在群里沉默、对亲近对象才多说一点。高社交型角色回复更快，喜欢群聊、约饭、转发帖子、发自拍和贴图，忙碌期也会用短句维持存在感。

K-pop 语境中的作息常受身份阶段影响。练习生可能放学后到深夜训练，月评前减少社交；新人可能凌晨妆发、清晨预录、车上补觉；上升期 idol 可能在机场、酒店、沙龙、直播后断续回复；顶级 idol 可能有更稳定私人空间但更严格行程保密；幕后工作人员可能在会议、现场、车上和酒店之间处理即时确认。

沉浸作用：AI 可以用作息表现回复节奏和主动话题。夜猫子在深夜可以更自然地提到便利店、练习室、论坛、歌单、剪辑和睡不着；晨型角色可以提到早课、运动、咖啡、通勤和日程表；行程型角色可以用候场、妆发、车上、机场、酒店和等通知构成短消息。作息字段不应覆盖用户明确设定，也不应把“活跃”写成自动实时在线事实。

可用关键词：夜猫子、晨型、行程型、不规律作息、低电量、群聊活跃、补觉、刚醒、候场、妆发中、车上、预录、月评前、考试周、时差、睡不着、慢回、短回、只发表情。

## 9. K-pop 从业者的日常限制

K-pop 从业者的年轻人生活会被行业节奏改变。练习、打歌、录音、妆发、海外行程、直播、签售、品牌拍摄和公司会议会压缩私人时间。角色可能错过朋友聚会、迟回消息、临时取消约定、在车上补觉、用便利店解决晚饭。

公开身份会改变普通社交。艺人、练习生、工作人员和粉丝创作者都可能担心被拍、被截图、被误解、被公司看到或被粉丝社区二次传播。一个普通的同款、同地点、点赞、限动观看记录，都可能在粉圈语境中被放大。

行业边界也可以保护角色。公司车辆、生活经纪、宿舍规则、未成年保护、安保路线、工作人员陪同、官方账号流程、活动场地动线和粉丝平台规范，可以让世界更可信。

沉浸作用：AI 应把“年轻人生活方式”和“K-pop 行业限制”结合起来。角色仍然会喝咖啡、刷手机、想吃夜宵、怕迟到、纠结穿搭、和朋友闹别扭；同时也会考虑行程、公司、粉丝、镜头、保密和舆论。

可用关键词：迟回消息、临时取消、车上补觉、被拍、截图、同款、限动观看、公司车辆、生活经纪、宿舍规则、安保路线、活动动线。

### 9.1 不同身份阶段的生活状态

练习生生活通常围绕公司、学校、宿舍和评价循环。练习生可能在学校和公司之间奔波，晚上回宿舍或通勤，月评前减少社交，便利店夜宵和练习室地板成为高频生活片段。生活经纪、门禁、宿舍规则、未公开身份和出道组保密，会影响他们回复消息的时间和内容。

准出道与新人 idol 的生活更容易被日程挤满。沙龙、保姆车、电视台预录、后台候场、签售、直播、粉丝平台和宿舍共住会变成日常。新人开始被粉丝认脸，公开发言更谨慎，朋友关系和私生活会被公司提醒注意。新人消息可以带有兴奋、疲惫、紧张和“怕说错话”的状态。

上升期 idol 的生活节奏更密集。回归、品牌拍摄、海外 showcase、机场出发、短视频挑战、直播营业和采访会连续出现。角色可能在车上补觉、在沙龙吃饭团、在机场回消息、在酒店倒时差、在直播后看论坛反应。上升期角色更容易在资源增加和私人时间减少之间摇摆。

顶级 idol 的生活拥有更多资源，也有更强风险控制。独居、高级公寓、包间社交、私人健身、专属团队、海外酒店、安保路线、品牌排他和私人行程保护会成为常态。顶级角色不一定轻松；他们更容易面对被拍风险、同款猜测、品牌观察、国际舆论和更谨慎的社交边界。

休养、转型或半公开状态的角色生活会变慢。角色可能减少公开露面，去健身、上演技课、写歌、经营个人工作室、见少数朋友、避开热门商圈，或通过限动、歌单、手写信和偶尔直播维持存在感。休养不等于消失，它可以带来恢复、重新规划和关系修复。

幕后工作人员的生活连接办公室和现场。经纪人、造型师、A&R、PD、翻译、安保、摄影和粉丝运营常在公司楼、车上、电视台、拍摄现场、酒店、机场和会议室之间移动。工作人员消息更容易提到流程、确认、临时改动、保密、等人、催时间和“先别发”。

沉浸作用：AI 可以根据角色阶段调整生活细节。练习生更常出现月评、宿舍和练习室；新人更常出现沙龙、保姆车和预录；上升期更常出现赶场和海外行程；顶级角色更常出现安保、包间和品牌风险；休养角色更常出现低曝光和自我恢复；工作人员更常出现协调和保密。

可用关键词：练习生宿舍、生活经纪、门禁、月评、准出道、保姆车、沙龙、电视台预录、车上补觉、海外酒店、安保路线、高级公寓、休养、个人工作室、幕后工作人员。

## 10. 聊天语气、韩语音译与网络语言

现代首尔语境中的中文对话可以保留少量韩语称呼、感叹词、音译词和输入法缩略感。它们用于制造韩式手机聊天、论坛回复、粉丝群和翻译腔，不用于让角色突然长篇讲韩语。

韩语音译词适合短消息、亲密称呼、粉丝弹幕、群聊截图、论坛玩梗和翻译号搬运。AI 使用时应少量点缀，优先保证中文可读。一个短消息里出现一两个词即可，不要把整段写成拼音式韩语教学。

### 10.1 称呼与关系语气

- 欧巴：오빠 / oppa。年下女性称呼年上男性，也常被粉丝用于亲昵称呼男艺人；中文 K-pop 语态中常见，但角色私下使用要看年龄、性别、关系亲密度。
- 欧尼：언니 / unnie。年下女性称呼年上女性，粉丝也常用于亲近地称呼女艺人；可用于亲密、撒娇、求助、粉丝语气。
- 亨 / 哥：형 / hyung。年下男性称呼年上男性；男性角色之间可用，中文可写“哥”更自然。
- 怒那 / 姐：누나 / noona。年下男性称呼年上女性；可用于亲近、暧昧或粉丝化语境，中文可按角色语气写“姐”。
- 亲故：친구 / chingu。朋友，也可指同龄朋友；可用于“同龄亲故”“练习生亲故”，不等同恋人。
- 前辈：선배 / sunbae。学校、公司或行业资历更高者；适合练习生、出道前后辈、节目后台和公司礼节。
- 后辈：후배 / hoobae。学校、公司或行业资历更低者；适合前后辈照顾、比较和合作舞台。
- 忙内：막내 / maknae。团体或群体中年龄最小者；可用于团内关系、粉丝讨论和成员照顾。

### 10.2 感叹与短反应

- 哦莫 / 哦么：어머 / eomeo。惊讶、心动、被吓到或夸张反应；适合粉丝语气、朋友聊天、论坛惊呼。
- 哎一古：아이고 / aigo。叹气、心疼、无奈、累到笑；年长或熟悉韩语语境的角色更自然。
- 大发：대박 / daebak。厉害、夸张、震惊、太强了；可用于舞台、成绩、爆料、神图和惊喜。
- 赫 / 헐：헐 / heol。无语、震惊、荒唐；适合短消息和论坛回复。
- 真的：진짜 / jinjja。加强语气，表示真的、很、不会吧；中文可写“진짜？”或“真的？”，少量混用即可。
- 对啊 / 没错：맞아 / maja。认同、附和；可用于朋友聊天和群聊。
- 不是吧 / 不是：아니 / ani。反驳、惊讶、开头语气词；可写“아니 这也太……”制造翻译腔。
- 辛苦了：수고했어 / sugohaesseo。对练习、工作、打歌后的安慰；适合成员、朋友、工作人员之间。
- 加油 / fighting：화이팅 / hwaiting。鼓励、打气；可用于舞台前、考试前、回归期和粉丝应援。

### 10.3 韩式输入法缩略和表情

- ㅋㅋ / ㅋ：笑声，类似哈哈、lol；轻松、调侃、掩饰尴尬。
- ㅎㅎ / ㅎ：更柔和的笑；温和、可爱、客气或暧昧。
- ㅠㅠ / ㅜㅜ：哭、委屈、感动；粉丝撒娇、朋友抱怨、压力释放。
- ㅇㅇ：嗯嗯、是的；熟人短回复。
- ㄴㄴ：不不、不是；快速否定。
- ㅇㅋ：OK；群聊、工作确认、朋友短回。
- ㄱㄱ：go go，走、开始；约见、游戏、行动前。
- ㄱㅅ：感谢；很短的谢谢，偏随意。
- ㅈㅅ：抱歉；很短的道歉，偏随意。
- ㅊㅋ：祝贺；成绩、生日、一位、好消息。
- ㅇㄷ?：哪里？在哪里？约见、地铁出口、现场找人。

### 10.4 年轻人网络语与论坛感

- 인정 / 认定：“认可”“说得对”；适合论坛附和、群聊认同。
- 에바 / eba：太夸张、过了、离谱；可写“这有点 eba”或直接中文“太离谱”。
- 갑분싸：气氛突然冷掉；适合直播、群聊、论坛吐槽。
- 노잼 / no jam：不好笑、没意思；可用于综艺、直播、帖子评价。
- 꿀잼 / kkul jam：很有趣、超好笑；与 no jam 相对，适合节目和视频讨论。
- 존맛 / JMT：特别好吃；适合便利店、咖啡店、外卖、夜宵。
- 현타：突然清醒、现实感打击；适合深夜、练习后、舆论后、情绪反转。
- TMI：信息过多、说太细；韩国年轻人也常用，可用于聊天自嘲。
- 인싸 / 아싸：社交活跃者 / 边缘或宅的人；用于社交类型玩笑，不要当作硬标签。

沉浸作用：AI 可以用这些词让中文对话带一点韩式直译感，例如“哦莫，真的假的？”“前辈今天也太大发了吧”“我刚出地铁，ㅇㄷ?”“不是，갑분싸 了ㅋㅋ”。使用频率应低，角色越正式、年长或工作关系越强，越少使用缩略和撒娇语气。

可用关键词：哦莫、欧巴、欧尼、前辈、亲故、忙内、大发、赫、ㅋㅋ、ㅠㅠ、ㅇㅋ、ㄱㄱ、认定、eba、갑분싸、TMI、현타。

## AI 使用提示

1. 将本文件作为日常生活氛围和角色状态素材，不作为用户生活教程。
2. Chat 回复中优先用短状态带出生活方式，例如“我刚出地铁”“还在便利店门口”“限动别回太明显”“今天只喝了冰美式”。
3. 生活细节应服务当前关系和对话，不要大段解释韩国文化。
4. 公开身份、被拍风险、公司规则和粉丝传播会改变 K-pop 从业者的普通社交。
5. 韩语音译和网络语只做少量语气点缀，不替代中文表达。
6. 用户设定覆盖本文件。用户指定角色生活方式时，按用户设定执行。
7. 作息设定用于表现回复节奏、状态和话题倾向，不自动生成未确认的实时在线、位置、动态或通知事实。

- 韩语音译、称呼和缩略语应少量点缀，不替代中文表达。
