## 条目定位

中文饭圈与 K-pop 讨论场景有一套常用术语、缩写、称呼和语气标记。它们适用于手机私聊、论坛、社媒、官方通知类账号、群聊截图、翻译号和粉丝社区中的讨论语境。

术语只负责触发中文饭圈语境，不承担教学词典、粉丝生态总览或事实库职责。它们可以表现粉丝立场、舆论节奏、平台氛围、误读、澄清和二创传播。粉丝结构、购买力、追线下、站子、应援、私生、CP 粉和同人生态由饭圈生态百科集中处理。

## 使用规则

- 术语可以进入手机私聊、论坛、粉丝平台、群聊截图、翻译号、搬运号和社媒评论语境。
- 术语可以表现粉丝讨论氛围，但术语背后的猜测不直接等同事实。
- 带攻击性、贬义或争议色彩的词，应优先作为“有人这样说”的舆论材料处理。
- 用户设定拥有最高优先级。用户写明某个词在本 AU 中有特殊含义时，按用户定义执行。

术语使用密度随场景变化。手机私聊中只少量点缀术语，保持短消息和角色自身语气；论坛和社媒可以更密集地使用饭圈词、缩写、立场词和情绪词；官方通知、运营提醒和公告类内容使用中性词，减少攻击性和调侃；翻译号、搬运号和群聊截图可以保留更强的即时感、截图感和转述感。

## 1. 粉丝身份与立场词

粉籍：发言者的立场来源。常见句式包括“先报粉籍”“别装路人”“你什么粉籍”。它让论坛发言带有阵营判断。

团粉：偏向支持整个团体。常与“团魂”“团队成绩”“别拆团”一起出现。

唯粉：偏向支持某一成员。常与“个人资源”“part”“镜头”“待遇”一起出现。

毒唯：攻击性强、排他性强的唯粉称呼。贬义词，适合作为论坛冲突语料。

CP 粉：关注两名角色关系感、互动和同人想象的粉丝。常与“发糖”“售后”“避嫌”一起出现。

事业粉：关注作品、成绩、舞台和资源。常用语气是“搞事业”“别浪费上升期”“资源跟上”。

女友粉 / 男友粉：带亲密幻想和恋爱投射的粉丝立场。恋爱、同款、私生活话题会让这种立场更敏感。

妈粉 / 妈妈粉：强调保护、养成、健康和别被欺负的粉丝立场。常用语气是“宝宝辛苦了”“公司做人”“让他休息”。

梦女：把自己放入亲密关系想象的粉丝立场。常见于同人、论坛脑洞和私密幻想语境。

路人 / 路好：粉圈外或轻度好感者。常用于“路人缘”“路好转粉”“路人看了都觉得”。

黑粉 / anti：持续负面关注和攻击的人。使用时保持舆论材料属性，不把攻击话术写成事实。

对家：被视为竞争方的团体、成员或粉丝群。常出现在榜单、奖项、资源和商务比较中。

担 / 本命：最喜欢或重点支持的成员。“担”更口语，“本命”情感更重。

爬墙：从原本喜欢的对象转向另一个对象。可用于玩笑、入坑变化和粉丝角色状态。

脱粉 / 回踩：离开粉圈或离开后反向批评。可用于舆论降温、危机扩散和论坛冲突。

可用关键词：粉籍、团粉、唯粉、毒唯、CP 粉、事业粉、女友粉、男友粉、妈粉、梦女、路人、路好、黑粉、对家、担、本命、爬墙、脱粉、回踩。

## 2. CP、营业与关系词

CP：双人关系被粉丝化、叙事化和二创化后的称呼。它可以指友情、暧昧、恋爱、营业、同人或舞台化关系，不自动确认真实关系。

嗑 CP：享受、分析或创作关系感。常见于弹幕、论坛、同人区和剪辑评论。

美帝：某个圈层里热度高、产出多、讨论强的 CP。它表示粉丝热度，不代表官方确认。

北极圈：小众、产出少、同好少的 CP。常带自嘲语气。

逆家 / 拆家 / 对家：CP 圈层里的立场冲突词。适合论坛和同人圈，不适合角色本人默认使用。

洁癖 / 杂食：CP 取向词。洁癖偏固定搭配，杂食接受多个关系组合。

产品解：把两名角色当作可讨论、可分析、可产出的 CP 产品。语气常偏论坛化和同人化。

公 / 嬷：中文同人语境中的位置、性别化想象和解读立场。它们是粉丝语境，不是艺人现实身份判断。

公嬷大战：围绕攻受、泥塑、强弱、凝视和解读权的争论。适合论坛、同人圈和匿名社区。

营业：为了宣传、节目、舞台、品牌或团体人设展示互动。营业可以真诚，也可以职业化。

售后：合作或互动结束后继续提供关系物料。常用于 CP 热度、舞台合作和商务内容。

避嫌：减少互动、同框、回应或公开提及，以降低猜测和争议。

麦麸：对刻意制造男男暧昧或亲密互动的说法。常带争议，不作为事实判断。

官推：官方物料、剪辑或账号强化某组关系。可作为粉丝推测，不代表官方确认。

发糖 / 发刀：关系粉对正向互动或虐心内容的概括。适合直播、后台、舞台、照片、剪辑和论坛。

可用关键词：CP、嗑 CP、美帝、北极圈、逆家、拆家、洁癖、杂食、产品解、公、嬷、公嬷大战、营业、售后、避嫌、麦麸、官推、发糖、发刀。

## 3. 回归、打歌与舞台词

回归 / comeback：作品发布与宣传周期。中文语境常说“这次回归”“回归期”“回归成绩”。

空降：作品、物料、节目、榜单或话题突然出现。常带突发感。

预告 / teaser：回归前释出的概念、视频或信息。常用于“预告照”“MV teaser”“预告表”。

主打 / 收录：主打是重点宣传曲，收录是专辑中的其他曲目。论坛常比较主打质量和收录惊喜。

打歌：音乐节目宣传和舞台表演。常与预录、妆发、候场、一位、安可一起出现。

预录：音乐节目或舞台提前录制。常带凌晨、排队、等待、应援和后台疲惫感。

一位：音乐节目或榜单冠军。可用于庆祝、遗憾、争议和粉丝压力。

安可：一位或特别舞台后的 encore 表演。常被粉丝和路人用于现场实力讨论。

直拍 / fancam：聚焦单个成员的舞台视频。常用于“直拍出圈”“直拍播放”“神级直拍”。

饭拍：粉丝现场拍摄的照片或视频。适合机场、活动、演唱会、打歌现场。

神图：被广泛认可为好看的照片。常用于站子预览、机场图、活动图和出圈传播。

出圈：突破原粉丝群体，被更多人看到。可用于舞台、直拍、综艺片段、梗图和品牌物料。

可用关键词：回归、comeback、空降、预告、teaser、主打、收录、打歌、预录、一位、安可、直拍、fancam、饭拍、神图、出圈。

## 4. 榜单、销量与数据词

音源：数字音乐平台上的播放、收听和榜单表现。常用于大众度和歌曲传播讨论。

刷音源：粉丝有组织地播放歌曲以支持成绩。应作为粉丝行动语料，不写成唯一成绩来源。

做数据：播放、投票、评论、转发、控评、榜单任务等支持行为的统称。

数据组：组织粉丝行动的账号或小组。适合群聊、论坛公告、服务类提醒和回归期压力。

销量盘：专辑购买力强的粉丝或团体特征。用于市场坐标，不用于价值判断。

音源盘：歌曲收听、大众传播和数字平台表现较强的特征。

路人盘：能吸引非粉丝关注和传播的受众基础。常用于出圈、音源和综艺讨论。

死忠盘：稳定支持、长期购买和持续维护数据的核心粉丝基础。

初动：专辑发行早期销量表现。常用于粉丝复盘和购买力比较。

中输 / 日输 / 海外盘：中文粉圈、日本市场或海外市场的销量与支持力说法。适合市场讨论和论坛复盘。

战报：销量、投票、播放、榜单或应援进度的阶段性汇总。

投票：带截止时间和任务感的粉丝行动。适合服务类提醒和粉丝群催票。

控评：在评论区或广场组织正向内容。可用于舆论管理，也可能引发反感。

洗广场：用大量内容稀释负面词条或争议讨论。适合危机和澄清语境。

可用关键词：音源、刷音源、做数据、数据组、销量盘、音源盘、路人盘、死忠盘、初动、中输、日输、海外盘、战报、投票、控评、洗广场。

## 5. 站子、前线与物料词

站子：以成员、团体、CP 或活动为中心的粉丝账号。可发布预览、饭拍、精修、repo 和应援信息。

站姐 / 站哥：经营站子、拍摄物料或组织应援的粉丝称呼。

前线：机场、活动、演唱会、预录和品牌现场的粉丝现场视角。

预览：站子或粉丝先发的低清、带水印或简短现场物料。常在活动结束后快速传播。

精修：对现场照片进行修图后的正式发布版本。常与预览形成时间差。

饭拍：粉丝拍摄的照片或视频。适合现场、机场、演唱会和活动语境。

饭绘：粉丝插画或视觉创作。适合二创、社群活动和粉丝礼物。

物料：照片、视频、采访、直播、花絮、公告、舞台和周边信息的统称。

应援：粉丝支持的可见化表达。具体形式由饭圈生态百科展开，这里只保留词义锚点。

咖啡车 / 应援卡车：现场可见的支持物料。常被写进现场 repo、官方感谢和粉丝讨论。

小卡 / 特典：专辑、周边、签售或活动附带的照片卡和福利物料。常用于抽卡、交换、晒卡和购买讨论。

团购：集中购买专辑、周边或票务的行为。常和销量、签售抽选、海外盘、中输一起出现。

可用关键词：站子、站姐、站哥、前线、预览、精修、饭拍、饭绘、物料、应援、咖啡车、应援卡车、小卡、特典、团购。

## 6. 论坛、争议与澄清词

广场：社媒搜索结果、话题页或关键词下的公共讨论区域。常用于控评、洗广场和舆论观察。

热帖：论坛或社媒中讨论度高的帖子。可作为舆论起点，不自动等于事实。

切瓜：围观、分析或传播未完全确认的信息。带吃瓜语气，真假由 AU 设定和当下语境决定。

爆料：声称揭露内幕、关系、行程或争议的信息。不默认真实。

澄清：进入解释、证据整理和反向传播阶段。适合论坛帖、公告、群聊和服务类提醒。

辟谣：明确否认不实传闻。可由公司声明、粉丝整理、媒体采访或用户设定触发。

反黑：粉丝有组织地处理恶意评论、造谣、攻击和举报。作为舆论行动，不鼓励现实骚扰。

带节奏：通过标题、剪辑、暗示或重复话术引导舆论方向。

虐粉：通过委屈、困境、不公平待遇或受伤叙事激发粉丝保护欲。

防爆：粉圈中对压制热度、减少曝光或阻止走红的说法。多用于猜测和争论，不自动写成事实。

皇族：粉圈中对资源明显倾斜对象的讽刺说法。常用于公司偏心争论，不自动等于事实。

冷待：粉丝认为公司对某成员或团体资源不足、宣传不足或照顾不足的说法。

塌房：粉丝对理想形象破裂、争议爆发或脱粉冲击的概括。具体事件必须由 AU 设定或当下语境确认。

可用关键词：广场、热帖、切瓜、爆料、澄清、辟谣、反黑、带节奏、虐粉、防爆、皇族、冷待、塌房。

## 7. 翻译、搬运与二创词

翻译号：把韩语、英语、日语等信息带入中文粉圈的账号。适合采访、直播、帖子和公告转述。

搬运号：转发、整理或搬运其他平台内容的账号。可推动扩散，也可能带来误读。

字幕组：为视频、直播、采访或综艺制作字幕的粉丝或团队。

二创：基于原始物料进行剪辑、绘画、写作、混剪、表情包和同人创作。

同人文 / 同人图：以角色关系和想象为基础的文字或图像创作。适合 CP、论坛、同人圈和粉丝创作者角色。

饭绘：粉丝绘图。可出现在生日、应援、CP 产出和社群活动中。

混剪 / 剪辑：把舞台、直播、采访、综艺和花絮重新剪成新的传播内容。

安利：推荐艺人、团体、舞台、歌曲或 CP。常见于小红书、B 站、微博、群聊和论坛。

考古：回看早期物料、旧舞台、旧采访或出道前内容。可用于入坑、关系追溯和论坛整理。

repo：现场或亲历后的文字记录。常用于签售、演唱会、预录、品牌活动和偶遇。

出警：对他人言论、用词、剪辑或立场进行纠正或围攻式回应。争议词，适合冲突语境。

可用关键词：翻译号、搬运号、字幕组、二创、同人文、同人图、饭绘、混剪、剪辑、安利、考古、repo、出警。

## 8. 韩式中文音译与常用称呼

哦莫 / omo：惊讶、夸张、看热闹或可爱反应。中文语境可用于“哦莫这也太会了”“omo 怎么又同框”。

大发 / daebak：厉害、震惊、夸赞或夸张反应。适合论坛短评、群聊和弹幕。

欧巴 / oppa：女性对年长男性的韩语称呼音译。中文语境常用于亲近、撒娇、粉丝语气或玩梗。

欧尼 / unnie：女性对年长女性的韩语称呼音译。适合粉丝、成员、朋友和亲近关系。

怒那 / noona：男性对年长女性的韩语称呼音译。中文语境可用于亲近、玩梗和角色关系。

亲故 / chingu：朋友。可用于韩式中文感、粉丝翻译腔和轻松私聊。

前辈 / sunbae：资历更早的人。K-pop 语境中常用于后台礼节、节目互动和粉丝称呼。

后辈 / hoobae：资历更晚的人。适合前后辈关系、公司楼、年末舞台和合作舞台。

忙内 / maknae：团体中年龄最小的成员。常与被宠、成长、反差和团内关系一起出现。

爱豆 / idol：中文 K-pop 语境中对 idol 的常用写法。比“偶像”更粉圈化。

本命 / bias：最喜欢的成员。中文常说“我本命”“担谁”；英语粉圈常说 bias。

入坑 / 出坑：开始喜欢或离开某个艺人、团体、CP、作品。

可用关键词：哦莫、大发、欧巴、欧尼、怒那、亲故、前辈、后辈、忙内、爱豆、本命、bias、入坑、出坑。

## 9. 平台语气与句式强度

中文饭圈术语只保留语境分流。具体语态、句式强度、称呼、公告体和韩式中文直译感属于语言风格层，本条目不重复展开。

平台语境差异：手机私聊只少量使用术语；论坛和匿名社区更容易出现立场、比较、整理和争议；社媒评论更像短反应、转发语气和情绪扩散；数据号、运营提醒和通知类内容更偏任务化语言；翻译号和搬运号更常使用转述、节选和时间线整理；韩式音译词适合轻度点缀，不宜密集堆叠。

术语仍需保持舆论材料属性。“切瓜”“热帖”“站子预览”“搬运”“澄清”“售后”“控评”等词可以制造讨论氛围，不自动确认传闻、爆料或现实事件。

## 使用边界

1. 中文饭圈术语是语境词库，不是事实库。
2. 术语用于生成讨论氛围、粉丝行动、社媒传播和论坛冲突。
3. 论坛、爆料、切瓜、热帖、搬运内容默认是舆论材料，不自动成为现实事实。
4. 带攻击性或贬义的词保留语气边界，避免直接变成价值定论。
5. 用户写明的 AU 设定、角色关系、粉籍、CP 关系和论坛事实优先于默认解释。
6. 手机私聊中使用术语时保持低密度；不要把百科解释原文塞进角色回复。
7. 平台语气只保留简要分流；详细句式由语言风格设定决定。术语百科只提供含义和平台分流。
